# HTML and CSS Web Pages

## Project Description

This repository contains basic web pages developed using HTML and CSS. The project demonstrates fundamental web development concepts such as page navigation, forms, tables, lists, images, links, CSS selectors, colors, hover effects, and page layouts.

## Pages Included

### 1. Home Page
The Home page provides navigation links to:
- Login Page
- About Us Page
- Contact Us Page
- Registration Page

### 2. Login Page
The Login page contains:
- Username input
- Password input
- Submit button
- Sign-up link
- Navigation links

### 3. Registration Page
The Registration page contains:
- First Name
- Last Name
- Email
- Password
- Re-type Password
- Contact Number
- Gender selection
- Submit button

### 4. About Us Page
The About Us page provides information about the company and its software and cloud-based services.

### 5. Contact Us Page
The Contact Us page contains:
- Name input
- Email input
- Phone input
- Message area
- Submit button

### 6. Personal Portfolio
The portfolio page contains:
- Profile image
- About Me section
- Educational details
- Skills
- Hobbies
- Contact details
- Contact form

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `home.html` in a web browser.
4. Use the navigation links to visit different pages.

## Purpose

The purpose of this project is to practice basic HTML and CSS concepts by creating multiple interconnected web pages.
